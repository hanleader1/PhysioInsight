# PhysioInsight
Chinese:<br>
本平台基于NeuroKit2开发，是一款可以对ECG、PPG、EDA、RSP、EMG信号进行处理与分析的工具。<br>
适用人群：生理信号的初学者或感兴趣的人。<br>
平台开源，未经允许不得用于盈利。<br>
Python解释器版本3.9,所需库（pip安装即可）：NeuroKit2，PyQt5等<br>
目录中Data为数据集，非强相关项。<br>
